# Dury
Dury is a simple note-taking app developed in Java, utilizing Spring Boot and the Java SDK for Android. The OCR folder is a plug-in feature for scanning text in images, powered by Tesseract.
# Tech Stack
- Java, Python
- Spring Boot, Flask
